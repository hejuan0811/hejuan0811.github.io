解决hibernate在不联网或者网很差的情况下不能解析文件

发表于 2016-05-06   |   分类于 jsp   |  

抛出的异常

原因

hibernate.cfg.xml配置文件中和映射文件XX.hbm.xml文件中解析需要到http:……的网站找XX.dtd文件，所以网不好或者网断了，就会抛出异常，不能进行解析

解决方法

在工程建一个文件夹，比如名为dtd的文件夹，在dtd文件加入hibernate-configuration-3.0.dtd和hibernate-mapping-3.0.dtd

jsp上传图片知识点

发表于 2016-04-28   |   分类于 jsp   |  

上传图片

  • 上传图片到服务器，通过组件上传，组件的包Smartupload，加入到工程
    代码如下：

    SmartUpload su = new SmartUpload();
su.initialize(getServletConfig(), request, response);
try {
    su.upload(); //上传，需要在前面
} catch (SmartUploadException e) {
    e.printStackTrace();
}
File img = su.getFiles().getFile(0);
String fileName = null;
if (img != null && img.getSize() > 0) {
    fileName = UUID.randomUUID().toString() + "." + img.getFileExt();//图片的名称
    try {
        img.saveAs("/upload/" + fileName);//保存到/upload文件夹下，在webroot目录下
    } catch (SmartUploadException e) {
        e.printStackTrace();
    }
}



    上传文件的页面，只需要在form表单下加入ENCTYPE属性就好
